# Validation fails with generic types.

When adding validation to minimal API, the source generation fails when any parameter uses an instantiated generic type.

## How to reproduce

Using version `10.0.100-preview.4.25258.110` of the dotnet sdk, run the following command in this folder:

```bash
dotnet build
```
